# Modelo Vosk para reconhecimento de voz (português)

Para o app **enviar com reconhecimento de voz pré-instalado** (sem depender de download na primeira abertura), coloque aqui o modelo pequeno em português:

1. **Baixe o modelo** (cerca de 45 MB):
   - https://alphacephei.com/vosk/models/vosk-model-small-pt-0.3.zip

2. **Coloque o arquivo** neste diretório, com o nome exato:
   - `vosk-model-small-pt-0.3.zip`

Não descompacte o zip: o app usa o arquivo `.zip` e extrai em tempo de execução.

Se este arquivo **não** estiver presente, o app fará o download na primeira vez que o usuário abrir (e guardará no aparelho). O modelo grande não é recomendado em celulares (falta de memória).
